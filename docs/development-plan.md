# 分阶段开发计划

## 阶段 1：项目初始化 + 前台基础页面

- 初始化 Next.js + TypeScript + App Router。
- 集成 Tailwind CSS 和 shadcn/ui 风格基础组件。
- 预留 Prisma + PostgreSQL 配置。
- 创建首页、产品中心、商品详情、订单查询、订单成功、文章、客服、政策和后台占位页面。
- 使用 mock 数据展示商品、文章和订单。

## 阶段 2：数据库模型 + seed 数据

- 设计 Product、InventoryItem、Order、Article、SiteSetting 等模型。
- 建立 Prisma migration。
- 编写 seed 数据脚本。

## 阶段 3：商品读取 + 下单流程

- 从数据库读取商品和库存状态。
- 实现商品详情页真实规格、数量和联系方式表单。
- 创建订单记录，但仍不接真实支付。

## 阶段 4：后台人工确认付款 + 自动发货

- 实现后台订单列表和订单详情。
- 管理员人工确认付款。
- 系统从文本库存中锁定并分配发货内容。

## 阶段 5：订单查询

- 实现订单号 + 联系方式查询。
- 展示支付状态、发货状态和发货内容。
- 补充售后联系入口。

## 阶段 6：SEO 文章系统 + AI 生成草稿

- 实现文章管理、分类、slug、meta、canonical 和 FAQ。
- 接入 AI provider 生成 SEO 文章草稿。
- 管理员审核后发布。

## 阶段 7：正式支付网关预留/接入

- 抽象支付订单和回调接口。
- 接入正式支付网关。
- 处理支付成功回调和订单状态流转。

## 阶段 8：安全加固和部署

- 增加后台认证、权限、审计日志和基础风控。
- 配置生产环境变量、数据库、部署流水线和监控。
- 完成性能、安全和可用性检查。
